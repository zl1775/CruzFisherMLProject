README

How to Run:

Produce wavelets and convolutions:
$ python3 wavelets1.py [FILENAME_LEFT] [FILENAME_RIGHT] [OUTPUT_DIRECTORY]

Example:

$ python3 wavelets1.py im0.png im1.png wavelet_img


Train binary classifier with window size 12:
$ python3 train.py 12

Test binary classifier with window size 12:
$ python3 test.py 12

Produce Linear Search Output:
$ python3 linear_search.py
